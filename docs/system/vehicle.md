# 自动驾驶车辆

一个自动驾驶系统包含以下组件：

- 传感器 (Sensors)
- V2X通信接口 (V2X communication interface)
- 感知 (Perception)
- 道路-世界模型 (Road-world model)
- 路径规划 (Path planning)
- 行为规划 (Behaviour)
- 短期路径规划 (Short-term path planning)
- 状态监测 (Health monitoring)
- 车辆线控系统 (Vehicle by-wire control)
- 数据记录单元 (Data logger)
- 嵌入式计算平台 (Embedded computing platform)
- 用户交互界面 (User interface)
