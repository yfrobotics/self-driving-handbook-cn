# 智能交通

智慧交通的核心技术是车联网 (V2X, Vehicle-to-Everything)，通过无线通信实现车辆与周围环境的信息交互，是实现协同自动驾驶和智慧交通系统 (ITS) 的基础设施。


## V2X 通信模式

| 模式 | 全称 | 说明 | 典型应用 |
| --- | --- | --- | --- |
| V2V | Vehicle-to-Vehicle | 车与车通信 | 前车紧急制动预警、协同变道 |
| V2I | Vehicle-to-Infrastructure | 车与基础设施通信 | 交通信号灯信息、限速预警 |
| V2P | Vehicle-to-Pedestrian | 车与行人通信 | 行人碰撞预警（手机端） |
| V2N | Vehicle-to-Network | 车与云端通信 | 远程监控、高精地图更新、OTA 升级 |


## 技术路线对比

目前有两条主要的 V2X 通信技术路线：

| 特性 | DSRC | C-V2X |
| --- | --- | --- |
| 全称 | Dedicated Short Range Communication | Cellular V2X |
| 标准 | IEEE 802.11p | 3GPP R14/R15/R16 |
| 通信模式 | Ad-hoc 直连 | 直连 (PC5) + 蜂窝 (Uu) |
| 典型延迟 | < 10 ms | < 10 ms (PC5)，< 100 ms (Uu) |
| 演进路径 | 标准成熟，演进有限 | 持续演进至 5G NR-V2X |
| 支持方 | 美国/欧洲部分厂商 | 中国、高通、华为 |

中国明确选择了 C-V2X 技术路线，并将其纳入国家战略。


## 应用场景

### 安全类
- **前向碰撞预警 (FCW)**：通过 V2V 获知前车异常减速
- **交叉路口碰撞预警 (ICW)**：检测侧向来车，减少鬼探头事故
- **弱势交通参与者预警**：V2P 感知视野外的行人和非机动车

### 效率类
- **绿波通行 (GLOSA)**：V2I 获取信号灯相位和时序，建议最优车速
- **协作式自适应巡航**：V2V 共享速度和加速度信息，实现车队紧密编队
- **动态路径规划**：V2N 获取实时路况，优化全局路线

### 信息服务类
- **高精地图众包更新**：车辆实时上报道路变化（施工、事故等）
- **远程驾驶 (Teleoperation)**：5G 低延迟网络支持远程操控无人车


## 中国 C-V2X 部署

中国在 C-V2X 产业推进方面处于全球领先：

- **频谱分配**：2018 年工信部划拨 5905–5925 MHz 用于 LTE-V2X 直连通信
- **路侧设备 (RSU) 部署**：在北京、上海、无锡、长沙等城市的测试区部署智能路侧单元
- **车路协同示范**：百度 Apollo、华为、大唐高鸿等参与车路协同项目
- **标准推进**：中国通信标准化协会 (CCSA) 和中国智能交通产业联盟 (C-ITS) 推动国标制定

未来趋势是结合 5G/6G 网络能力，从单车智能向"聪明的车 + 智慧的路"的车路协同方向发展。
