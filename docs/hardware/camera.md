# 视觉传感器

摄像头是自动驾驶中最重要的传感器之一，能够提供丰富的颜色、纹理和语义信息。本节介绍车载摄像头的类型、关键参数和典型布置方案。


## 摄像头类型

| 类型 | 特点 | 典型应用 |
| --- | --- | --- |
| 单目摄像头 (Monocular) | 成本低，无法直接测距 | 车道线检测、交通标志识别 |
| 双目摄像头 (Stereo) | 通过视差计算深度，精度受基线影响 | 近距离障碍物测距 |
| 鱼眼摄像头 (Fisheye) | FOV 可达 180°–190°，畸变大 | 环视泊车系统 |
| 红外摄像头 (Infrared) | 感知热辐射，不受光照影响 | 夜间行人检测 |
| 事件相机 (Event Camera) | 异步输出像素亮度变化，时间分辨率极高 | 高速场景、HDR 场景 |


## 关键参数

| 参数 | 说明 | 典型范围 |
| --- | --- | --- |
| 分辨率 (Resolution) | 像素数量，影响识别距离 | 1–8 MP |
| 帧率 (Frame Rate) | 每秒采集帧数 | 30–60 fps |
| 视场角 (FOV) | 水平/垂直可视范围 | 30°–190° |
| 动态范围 (Dynamic Range) | 明暗场景适应能力 | 90–140 dB |
| 快门类型 (Shutter) | 全局快门消除运动模糊，卷帘快门成本低 | Global / Rolling |


## 车载摄像头布置

典型 L2+ 自动驾驶车辆配备 6–12 个摄像头，覆盖 360° 视野：

- **前视主摄像头**：窄 FOV（约 30°–50°），长焦，用于远距目标检测
- **前视广角摄像头**：宽 FOV（约 120°），用于近距离交叉路口感知
- **侧视摄像头**：左右各一，覆盖盲区和变道场景
- **后视摄像头**：倒车影像与后方来车检测
- **环视鱼眼摄像头**：四角各一，组成鸟瞰拼接图 (BEV)，用于泊车


## 行业方案

- **Tesla 纯视觉方案**：依靠 8 个摄像头（后改为 7 个）+ HW3/HW4 芯片，通过 Occupancy Network 和 BEV 感知实现端到端自动驾驶，不依赖 LiDAR
- **Mobileye**：EyeQ 系列芯片搭配单目/多目摄像头，广泛用于 L2 级 ADAS (Advanced Driver Assistance Systems)，提供车道保持、自动紧急制动等功能
- **大疆车载**：基于立体视觉方案，结合惯导实现高性价比感知
